## Instrucciones variables de ambiente

Para que puedan leer datos de Minio es necesario seguir las siguientes instrucciones:

- Crear un archivo .env en el origin del proyecto
- Pegar las siguientes variables de autenticación:
  - ACCESS_KEY
  - SECRET_KEY
  - AWS_S3_ENDPOINT
